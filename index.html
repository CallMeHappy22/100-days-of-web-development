HTML & CSS Basics Summary

HTML Summary

HTML (HyperText Markup Language) is used to define our page content,structure and meaning. You don't use it for styling purposes. Use CSS for that instead!

  • HTML uses "elements to discribe (annotate) content
  • HTML elements typically have an opening tag, content, closing tag
  • You can also have void (emoty) elements like images
  • You can configure elements with attribute

Learn more about all available HTML elements on the MDN HTML elements reference.

CSS Summary

CSS (Cascating Style Sheets) is use for styling your page content.

  • Styles are assignef via property-value pairs
  • You can change assign styles via the "style" attribute
  • To avoid code duplication, you typically use global styles (eg. via the "style" element) instead
  • Alternative, you can work with external stylesheet files which you "link" to
  • When working with CSS, concept likw "inheritance","specificity" and "the box model" shoule be understood.

Learn more abour all available CSS properties and value on the MDN CSS property sreference
